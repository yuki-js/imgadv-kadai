## 前提

2クラスのデータセットを用意した。
- AokiAppデータセット: 青木の顔画像データセット
- Itadakimasu Man NFTデータセット: いただきますマンの顔画像データセット(MITライセンス)

datasetsディレクトリにそれぞれnpy形式で保存されている。各データセットは高次元ベクトル(画像を1次元配列に変換したもの)の集合として表される。

現行のデータセットはいわゆるtrainとtestに分かれていない。部分空間を構成するtrain(厳密にはtrainと評するべきではないが)、および評価用のtestデータセットを別途分ける機能を搭載すること。

可視化の際にはGUIを用いてインタラクティブに回転・拡大縮小できるようにすること。3次元プロットにはPlotlyを用いること。

## Query 1
２クラスの高次元ベクトルデータセットをそれぞれ３次元の部分空間S1とS2で表し，両者の成す正準角セットを求めよ．

## Query 2
S1とS2の差分部分空間を求めよ．

## Query 3
2で得られた差分部分空間の基底ベクトルを画像として可視化せよ．

## Query 4
各クラスのデータを差分部分空間へ射影して３次元プロットせよ．

## Query 5
１～３に基づいて，差分部分空間の特性について様々な観点から考察せよ．
