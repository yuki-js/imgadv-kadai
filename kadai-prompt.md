## 前提

2クラスのデータセットを用意した。
- AokiAppデータセット: 青木の顔画像データセット(自己の著作物) aoki.npy
- Itadakimasu Man NFTデータセット: いただきますマンの顔画像NFT(MITライセンス)から作成したデータセット itadakimasu.npy

datasetsディレクトリにそれぞれnpy形式で保存されている。各データセットは顔画像を一度 32x32x3 に縮小し、これを1次元配列に変換した高次元ベクトルの集合として表される（前処理で縮小を必ず行うこと）。

現行のデータセットはいわゆるtrainとtestに分かれていない。部分空間を構成するtrain(厳密にはtrainと評するべきではないが)、および評価用のtestデータセットを別途分ける機能を搭載すること。
そして、データセットサイズの数のバランスが悪いので、多いほうを少ないほうにそろえよ。

可視化の際にはGUIを用いてインタラクティブに回転・拡大縮小できるようにすること。3次元プロットにはPlotlyを用いること。

これは、部分空間法、特に差分部分空間法に関して、構築と様々な可視化を通じて理解を深めるための課題である。部分空間法に関する深い理論的背景を授業で学んでいることを前提とする。

## ツールキット

subspace-tkに、部分空間法のリファレンス実装がある。参考にしてほしい。ただし、直接の利用は避け、自分で一から実装すること。

## Query 1
２クラスの高次元ベクトルデータセットから３次元の部分空間S1とS2を構築し，両者の成す正準角セットを求めよ．

## Query 2
S1とS2の差分部分空間を求めよ．

## Query 3
2で得られた差分部分空間の基底ベクトルを画像として可視化せよ．

## Query 4
各クラスのデータを差分部分空間へ射影して３次元プロットせよ．

## Query 5
１～３に基づいて，差分部分空間の特性について様々な観点から考察せよ．
