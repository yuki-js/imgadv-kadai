## 1. 評価データに関する説明

本研究では、暗号資産（仮想通貨）の時系列価格データに対して主成分分析（PCA）を適用し、市場全体の共通要因とセクター的・銘柄固有の変動要因を抽出・解釈した。解析対象のデータセットは、Binance の現物市場における USDT 建ての高流動銘柄から上位を選定し、過去3年間の「日次終値」により構成した（行＝銘柄、列＝日付）。データ収集・ユニバース定義・解析は、以下のパイプラインで実装した（収集: requests、加工: pandas、PCA: scikit-learn、可視化: matplotlib/seaborn）。

ユニバース選定の基準は次の通りである。  
- 現物市場の USDT 建てで、ステータスが TRADING、かつ spot 許可がある銘柄を対象とした。   
- 安定通貨・疑似安定通貨やラップト資産は除外した（例: USDT/USDC/FDUSD/DAI/USDP/PAX/SUSD、WBTC/WETH/WBETH など）。また USDE はアルゴリズム型安定通貨として除外した。  
- 「3年前に上場済みであった」銘柄に限定した。 

データは Binance API から日次足（1d）の終値を取得し、収集期間（開始日〜終了日）で欠損を含むものは前後から按分した。日次終値系列は data/crypto_prices_3y.csv に保存し、生値（raw price）を使用する。以後は「日付（サンプル）×銘柄（特徴）」行列に転置し、各特徴（銘柄）を標準化（平均0・分散1）した上で PCA を適用する。この方法により、非線形変換（対数など）を排除し、線形処理のみで主成分を抽出することで、価格そのものの時系列構造を保持しつつ、銘柄間のスケール差を標準化により吸収している。

## 4. 考察

まず、寄与率の分析を行う。PC1は45%という高い寄与率を示し、市場全体の共通要因の存在が強く示唆される。アルトコインはBTCの価格変動に強く影響を受けるのが常であることから、これは妥当な結果である。PC2も比較的大きい寄与率を持ち、34%であった。PC1とPC2を合わせると79%の寄与率となり、これら2成分で市場全体の大部分の変動を説明できることがわかる。これより、暗号資産市場は非常に高い相関構造を持ち、長期トレンドは非常にわかりやすいものとなっていることが示された。PC3以降は寄与率が急激に低下し、PC3で5.7%、PC4で4.8%、PC5で2.1%となっている。PC3以降は銘柄やカテゴリ固有の変動要因や短期トレンドと考えられる。

PC1-3 において、2024年1月、2024年11月、2025年2月に大きく変動している。 これらは市場全体の大きなトレンド転換点に対応している。
2024年1月はビットコイン半減期とビットコインETFの承認が重なり、BTCを中心に大きく相場が上昇した。
2024年11月はアメリカ大統領選挙において暗号資産推進派のドナルド・トランプ氏が再選された。
2025年2月はドナルド・トランプ氏が大統領に就任し、それに伴い多くの大統領令に署名し、その中に暗号資産に関するものも含まれていた。また、就任前日にドナルド・トランプが発行したミームコイン「$TRUMP」により相場が過熱した。
このようなニュースによって、BTCはじめ多くの暗号資産が大きく変動した時期であった。
